Python 官方推荐的学习书单

2020-12-15 5 min read

Python官网书单为Python学习者提供了各种类型的学习书籍，不过都是英文的，如下图。

点击进入，可以看到该页面提供三十本编程初学者书籍。

不过考虑到大家可能没时间看这些，或者只是简单学习一下，不是专业工程师，所以我从中挑选几本比较经典的，找到中文版都下下来。

按照学习路径排序。

入门

第一本肯定是最经典的Python编程：从入门到实践，在官网初学者书单排名也是排名第一。虽然放在入门书籍类目中，对于有编程经验的同学，也可以用来系统回顾知识。本书分为两个部分，首部分介绍Python编程基本概念，第二部分将理论转化为实践，包括三个项目。如果觉得吃力，可以先看下面两个。

第二本简明Python教程，这是一本国内翻译的手，在Github上不断更新，豆瓣评分8.8。最简洁易懂的Python书了，超级基础，就像小时候老师手把手教你1+1=2，就像书的名字一样。

第三本Python学习手册，这是一本官方初学者书单的第三名。相比较前两本，这一本更加全面深度的介绍了Python语言的核心知识。

最后一个就是官方文档，官方文档就相当于一个产品说明书，在这里就是Python这门语言的说明。

下载链接：入门书籍下载

进阶

第一本同样是官方推荐流畅的Python，这本书是深入理解Python的好书之一，让你更平滑的从入门到进阶，豆瓣评分9.5。

第二本Python Cookbook，本书覆盖了Python应用中的很多常见问题，并提出了通用的解决方案。书中的代码和方法具有很强的实用性，可以方便地应用到实际的项目中，并产生立竿见影的效果。豆瓣评分9.3。

第三本Python高手之路，虽然本书只有两百多页，但是本书内容十分丰富，还涉及很多高级主题，如性能优化，插件优化等。

下载链接：进阶书籍下载

爬虫

Phthon3网络爬虫开发实战，国内崔庆才老师写的，很基础很实用，豆瓣9.0分，老师是微软的哦。

一本书看完差不多了，最重要的是要多敲多练，GitHub上有很多爬虫案例，在练习的时候可以clone下来，修改修改，跑一跑就有感觉了。

图解HTTP，作者由HTTP协议的发展史娓娓道来，严谨细致地剖析了HTTP协议的结构，列举诸多常见通信场景及实战案例，最后延伸到Web安全、最新技术动向等方面。了解了网络知识，有助于了解爬虫中的网络问题。

《Head First HTML与CSS》，这本是前端相关的基础知识，因为在爬取页面时，总的调出F12找到相应的元素，所以前端知识不能少。本书内容幽默有趣，很容易看下去。虽然很多人说是入门书，但工作多年的前端也推荐阅读。

SQL基础教程，爬取数据之后，总要存起来，掌握一点SQL基础，方便管理数据。

下载链接：爬虫书籍下载

数据分析

集体智慧编程，以具体实例的方式来展示 Python 的编程技巧，非常直观的向读者展示了人工智能和机器学习中的大量经典的算法。

更可贵的是，作者在展示算法时所使用的例子都是网络中非常有代表性的场景，并且很多情况下还会结合一些实际运营的 Web 站点的数据作更进步阐释。

就是说这是一本可以拿来抄作业的书籍。豆瓣评分9.0

利用 Python 进行数据分析，本书由Python pandas项目创始人Wes McKinney亲笔撰写，详细介绍利用Python进行操作、处理、清洗和规整数据等方面的具体细节和基本要点。
